Componentes

Cliente




dispositivo o aplicación que solicita servicios. Puede ser una computadora, un teléfono inteligente o un software como un navegador web.

Servidor:




Máquina o programa que administra y responde a las solicitudes de los clientes. Puede ofrecer servicios como bases de datos, aplicaciones, almacenamiento de archivos, entre otros.

Red:




Medio físico y lógico a través del cual se establece la comunicación. Puede ser una red local (LAN) o una red global como Internet

Protocolos:




conjunto de reglas que permiten el intercambio de información entre cliente y servidor, como HTTP, TCP/IP, DNS, entre otros.
