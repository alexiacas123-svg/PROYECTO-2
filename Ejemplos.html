Ejemplos de uso en la actualidad

El modelo cliente-servidor está presente en múltiples aplicaciones y servicios tecnológicos:


  • Navegadores web: los clientes (navegadores) solicitan páginas a los servidores web mediante HTTP o HTTPS.
  • Correo electrónico: los clientes de correo como Outlook o Gmail acceden a servidores mediante protocolos como SMTP, POP3 o IMAP.
  • Bases de datos: las aplicaciones empresariales consultan información almacenada en servidores de bases de datos como MySQL, Oracle o SQL Server.
  • móviles: la mayoría de las apps de mensajería, banca y redes sociales se basan en el modelo cliente-servidor.
